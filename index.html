Description

This course is an introduction to programming using Swift, Apple's new programming language for Macs and iOS devices. Through Swift, we explore the basics of programming and introduce fundamental computer science topics like abstraction, algorithms, basic data structures, and recursion. The course also features practical programming topics such as scripting, command line interfaces, application programming interfaces, debugging, integrated development environments, understanding documentation, and developing software both in isolation and as a team. By the end, each student designs and implements an app that can be deployed to iOS devices, like an iPhone or an iPad. Students must bring a MacBook, MacBook Air, or MacBook Pro running Mac OS10.11 or higher.

Day 0

Abstraction, algorithms, pseudocode, source code, compilation and interpretation. Swift basics: variables, boolean expressions, logical operators, arithmetic, and control flow.

Homework Due

  1. Install OS X 10.11.5 and Xcode 7.3.1 for class
  2. Fill out pre-survey
  3. Sign up for a free Apple Developer Account (do not sign up for the $100 account!)

Day 1

Theory of computation, programming paradigms. More Swift topics including data types, arrays, and functions.

Day 2

Algorithmic complexity, parallelism, version control. More Swift topics including data types, arrays, and functions.

  • Eliot: Loops, exercise review, how to approach looping problems.
  • Kirkland: Version control and Git.

Day 3

Functions, dictionaries, closures, lambda. Possibly: defining new operators and higher-order functions.

Day 4

Binary, ASCII, basic encryption, iOS.

Day 5

Game of Fifteen, Xcode, iOS application lifecycle.

Day 6

More details on Storyboard elements, including constraints. Programmatically creating UI elements.

Day 7

Bounds and frames of UI elements, animating UI element properties, timers, debugging (via breakpoints and print statements), more on approaching the Game of Fifteen app.

Day 8

UI alerts with one and two buttons, action sheets with multiple buttons. Alternative approaches to the Game of Fifteen implementation.

Day 9

Wrap-up.
